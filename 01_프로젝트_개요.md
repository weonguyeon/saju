## 🏗️ 프로젝트 구조

```
프로젝트_폴더/
├── .env                    # 환경변수 (OpenAI API 키)
├── app.py                  # Flask 메인 애플리케이션 (100줄)
├── ai_analysis.py          # OpenAI GPT-4o 연동 모듈 (83줄)
├── saju_logic.py           # 사주 계산 핵심 로직 (705줄)
├── requirements.txt        # Python 패키지 의존성
├── static/
│   └── style.css          # 글로벌 스타일 (506줄)
└── templates/
    ├── index.html         # 입력 폼 페이지 (59줄)
    ├── loading.html       # 로딩 애니메이션 (141줄)
    └── result.html        # 결과 표시 페이지 (405줄)
```

---

## 🔄 사용자 플로우

```
[1. index.html] 입력 폼
       ↓ POST /loading
[2. loading.html] 로딩 애니메이션 (300ms 후 자동 제출)
       ↓ POST /result
[3. result.html] 4개 탭으로 결과 표시
       ↓ "다시 입력하기" 클릭
[1. index.html로 복귀]
```

### 상세 플로우

1. **입력 페이지**: 이름, 성별(라디오), 생년월일(양력), 태어난 시간
2. **로딩 페이지**: 신비로운 스피너 + 5개 순환 메시지, 숨겨진 폼으로 300ms 후 자동 제출
3. **결과 페이지**: 
   - 사주 원국표 (5열 테이블: 구분/시주/일주/월주/년주)
   - 4개 탭: 기본해석, 오행&십성, 인생의흐름(근묘화실), 대운(10년×8주기)

## 🚀 실행 방법

```bash
# 1. 패키지 설치
pip install -r requirements.txt

# 2. .env 파일에 API 키 설정
echo "OPENAI_API_KEY=sk-your-key" > .env

# 3. 서버 실행
python app.py

# 4. 브라우저 접속
# http://127.0.0.1:5000
```
